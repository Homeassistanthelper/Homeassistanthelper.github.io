Home Assistant Einrichtungsanleitung

Schritt 1: Paketzustellung

Vereinbaren Sie einen Termin mit einem unserer besten Helfer. Ihr Paket wird mit ihrem geliefert.

Schritt 2: Paket öffnen

Öffnen Sie vorsichtig die Verpackung und nehmen Sie den Raspberry PI heraus.

Schritt 3: Zigbee-Antenne

Drehen Sie die Zigbee-Antenne auf das Zigbee-Modul.

Schritt 4: Schließen Sie die Zigbee-Antenne an

Stecken Sie die Zigbee-Antenne in den USB-Anschluss des Raspberry Pi 4

Schritt 5: Stecken Sie das Netzkabel ein

Stecken Sie das Netzkabel in eine Steckdose und in den Computer (Raspberry Pi 4).

Schritt 6: Stecken Sie das Ethernet-Kabel ein

Schließen Sie ein Ethernet-Kabel an den Computer (Raspberry Pi 4) an und verbinden Sie diesen zu Ihrem Router. Das Ethernet-Kabel ist nicht im Lieferumfang enthalten, Sie müssen sich selbst eines besorgen.

Schritt 7: Letzter Schritt

Gehen Sie weiter zur Webseite homeassistant.local:8123. Sie sehen nun Ihren Server.

Schritt 8: Fertig

Jetzt ist Ihr HomeAssistant einsatzbereit. Sie können jetzt Geräte hinzufügen und automatisieren.
